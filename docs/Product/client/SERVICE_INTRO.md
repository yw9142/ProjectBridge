# Bridge Client 서비스 소개

Bridge Client는 발주/클라이언트가 프로젝트 진행 상황을 확인하고 승인/응답하는 작업공간입니다.

## Client 핵심 기능

1. 내 프로젝트 확인
2. 요청 상태 확인/응답
3. 파일 검토
4. 회의 참석 응답
5. 계약 검토(승인/반려)
6. 정산 상태 확인
7. Vault 정보 확인
8. 전자서명 페이지 접근 및 서명 제출
