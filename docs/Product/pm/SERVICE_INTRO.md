# Bridge PM 서비스 소개

Bridge PM은 벤더가 프로젝트 실행을 주도하는 실무 작업공간입니다.

## 핵심 기능

1. 프로젝트 생성/운영
2. 게시글 기반 커뮤니케이션
3. 요청 상태 관리
4. 파일 버전/리뷰 협업
5. 회의 일정 운영
6. 계약 및 전자서명 진행
7. 정산(Billing) 관리
8. Vault 계정 정보 운영
