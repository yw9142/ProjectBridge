# Bridge Admin 서비스 소개

Bridge Admin은 플랫폼 운영자가 테넌트, 사용자, 프로젝트를 통합 관리하는 콘솔입니다.

## Admin 핵심 기능

1. 테넌트 생성/상태 확인
2. 테넌트 상세 조회(프로젝트, PM 사용자 포함)
3. 프로젝트 단위 대시보드 모니터링
4. 운영 관점의 사용자/권한 관리
